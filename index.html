Инструкция по созданию бота

1

Регистрируем бота в Botfather

  • Открой @BotFather в Telegram и нажми Start.
  • Введи команду /newbot.
  • Скопируй API Token — он нам пригодится.
2

Создаем проект в Salebot

  • Зайди на платформу и создай новый проект.
  • Примени промокод для 10 дней бесплатного доступа.
NAZARINA
3

Первое сообщение бота

  • В «Конструкторе воронок» создай блок «Стартовое условие».
  • Напиши приветствие для твоих клиентов.
4

Связываем бот с воронкой

  • В разделе «Каналы» выбери Telegram.
  • Вставь токен и нажми «Запустить».
5

Добавляем кнопки

  • Нажми «Добавить кнопку» в первом блоке.
  • Выбери Inline-тип для современного вида.
6

Логика переходов

  • Создай блок с типом "Не состояние с условием".
  • Выбери соответствие "Полное совпадение".
7

Команды меню

  • Настрой меню через /mybots в BotFather.
  • В Salebot создай блок, реагирующий на команду.
